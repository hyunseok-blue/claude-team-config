# 작업 완료 시 문서 업데이트

작업 중 프로젝트 내 문서, skill, agent를 참조했다면, 작업 완료 시 해당 문서도 함께 업데이트한다.

## 대상
- **Agent 정의** (`.claude/agents/*.md`): 새로운 파일/엔드포인트/패턴이 추가되었으면 Key Files, API Endpoints, Rules 등 해당 섹션 반영
- **Agent Memory** (`.claude/agent-memory/*/MEMORY.md`): 작업 중 발견한 패턴, 에러 해결, 주의사항 기록
- **Skill 정의** (`.claude/skills/*.md`): 워크플로우나 위임 전략이 변경되었으면 반영
- **프로젝트 CLAUDE.md**: 프로젝트 구조, Tech Stack, 컨벤션 등이 변경되었으면 반영
- **설계 문서** (`docs/`): 아키텍처, 인프라, DB 스키마 등 관련 문서가 있으면 반영

## 원칙
- 코드 변경과 직접 관련된 문서만 업데이트 (불필요한 문서 수정 금지)
- 문서 업데이트는 작업의 마지막 단계로 수행
- 변경 내용이 사소하면 (오타 수정, 포맷 변경 등) 문서 업데이트 생략 가능
- 문서 업데이트 시 기존 구조와 스타일을 유지
